This is the official dataset for GraphRAG-Bench: Challenging Domain-Specific Reasoning for Evaluating Graph Retrieval-Augmented Generation.

It contains 5 question types spanning 16 disciplines and a corpus of 7 million words from 20 computer science textbooks.
